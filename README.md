## Setup/Installation Requirements
* Clone the repository to your local machine
    ```
    $ git clone https://github.com/Derraphics/My-Portfolio-.git
    ```
* Move into the cloned directory
    ```
    $ cd My-Portfolio
    ```
* Open the project on VS Code and liveserver
